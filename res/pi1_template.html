This template contains the HTML used when the Facebook Connect extension is inserted into a page. The normal logout-button must be wrapped in Logout See http://wiki.developers.facebook.com/index.php/JS_API_T_FB.Connect, look for logout() and logoutAndRedirect(). The Javascript needed on every page where we need Facebook magic The Facebook Connect button See different styles at http://wiki.developers.facebook.com/index.php/Fb:login-button Profile picture and user name - inserted in templates below

Info shown when autologin fails (no FE user, but a connected FB user) ###PROFILE###

Your Facebook account is not linked to a website user.
Please use the normal login and then link your accounts.

Info shown when an FE user with no facebook link is logged in and no Facebook user is connected

If you have a Facebook account, you can connect it to your website user.

###CONNECT_BUTTON### Info shown when an FE user with no facebook link is logged in and a Facebook user is connected ###PROFILE###

If you link your Facebook account to your website account you can log in using Facebook Connect.
Link accounts

Info shown when the logged in FE user is linked to a Facebook user not currently connected.

Your website user is linked to your Facebook account, but you are not currently connected.
You can break the link if you wish: Unlink accounts

###CONNECT_BUTTON### Info shown when the logged in FE user is linked to the connected Facebook user ###PROFILE###

You can break the connection between your Facebook account and the website user.
Unlink accounts

Info shown when the logged in FE user is linked to another Facebook user than the connected Facebook user ###PROFILE###

Your website user is linked to another Facebook account.
Unlink accounts
